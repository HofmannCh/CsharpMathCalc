 # Windows Math Calculator 
 After executing, global hotkey listener `Control + Y` opens calculator, where you can easily calculate Math expressions and save it to clipboard.
 You can call simple functions like `guid()` with gives you a random guid.
 At the moment it seems that global hotkey listener for Windows 10 is broken and no status for Windows 11, because I had to change working machine.
